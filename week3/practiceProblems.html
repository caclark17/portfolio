Week 3 Notes and Practice

This

A function that is a property of an object is called a method.

To access an object a method can use the "this" keyword.

Arrow functions don't have their own "this."

Objects

  • An object is a self-contained set of values and functions like an array
  • If the value of a property is a function then it's called a method
  • An object literal is wrapped in curly braces and contains all of it's properties and methods
  • All objects are mutable even when const is declared.
  • To call and object you can use dot or bracket notation.

    • superman.fly() OR superman['fly']()

  • An object is like a dictionary where you can see a value when you look up a property.
  • Calling a method is the same as invoking a function and can be written two ways:

    • superman.fly()

    superman['fly']()

  • To loop through the properties and methods of an object:

    • for(const key in superman) {

    console.log(key + ": " + superman[key])'

    }

  • You can remove a property by using the delete operator. Example: delete superman.fly

    • How often does a programmer actually use this? Why not just erase it from the code?

  • An object can contain other objects.

    • When would I want to use nested objects as opposed to an object literal?

  • Object literals can be passed as a parameter to a function.
  • Placing functions inside an obejct can help to prevent problems if you reuse a variable without realizing it. This is called namespacing.
  • JSON data can be easily turned into a JS object but using the parse() method.

    • Example: JSON.parse(batman);

  • A JS object can be turned into JSON by using stringify().

    • Example: JSON.stringify(wonderwoman);

Document Object Model

  • Document.body returns the body element of the page.
  • Document.images returns a node list of all the images.
  • Document.links returns a list of a elements and area elements that have an href attribute.
  • Document.anchors returns a list of all the a elements that have a name attribute.
  • Document.forms returns a list of all the forms in the document.
  • You can turn a node list into an array by using Array.from().
  • getElementsByTagName() will give you a list of all the elements that have the tag name of what you put in as an argument.
  • By using heroes.insertBefore(aquaman, wonderWoman); then aquaman would be added before wonderWoman in the list.

Events

  • An event listener lets the user know when a certain event has happened.
  • Other types of events include mouse events, keyboard events, and touch events.
  • When an event listener is added to a parent it is called event delegation.
  • Some events can happen when a user clicks on a certain key on the keyboard.
  • A touchstart event happens when the user touches the surface of something like a touchscreen or track pad.
  • A touchend event is when the user stops touching the surface.
  • Event propogation is the order that events are fired.
