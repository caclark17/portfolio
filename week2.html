Chapter 2: Programming Basics

  • Something I forget to utilize is the "typeof" operator. Example:

    • typeof 10

    returns 'number'

  • Another thing I forget to utilize is declaring multiple variables at the same time. Example:

    • let x=3, y=4, z=5;

  • Local scope means a variable only exists within the block that it's declared in
  • Global scope means a variable can be used anywhere within a program, but you need to be careful because it can cause problems if you forget you already used it and then try to use it again.
  • name.toUpperCase() will change this variable to make all the letters caps
  • name.includes('a') will tell us if that letter is included in that string.
  • name.startsWith('A') is case-sensitive.
  • You can convert a number to a string by using String(3) (replace 3 with whatever number you want to use.

Chapter 3: Arrays, Logic, and Loops

  • Initialize an array called fruit:

    • const fruit = [];

    Add a value by putting in the number of the position where you want it placed:

    fruit[1]= banana;

    fruit[0]= apple;

  • The join method can turn an array into a string. Example:

    • fruit.join();

  • fruit.splice(5,0,'orange'); Would put orange at position 5 and not remove any items.
  • You can reverse the order of an array by using the reverse() method.
  • You can put the contents of an array in alphabetical order by using sort().
  • Using indexOf() will tell you the index location of an item in an array.
  • Sets are a collection of unique values.
  • A value can be placed in a set using the add() method.
  • Use the delete() method to remove a value from a set.
  • Maps keep a list of key and value pairs.
  • Using clear() will remove all keys and values from a map.
  • Maps can be converted to arrays using Array.from()

Chapter 4: Functions

  • Functions can be entered like a string such as:

    • const hi = new Function('console.log("Hi World!");')

    This is not recommended though.

  • Functions have to be called in order to run. Make sure to use the function name with parenthesis when calling.

    • Parameters and Arguments

    • Parameters are set when a function is defined.
    • Arguments are provided when the function is invoked.

      • Example:

      function square(x) {return x*x; }

      square(4.5);

      returns 20.25

    • Default parameters can be used when no arguments are provided.
    • Arrow functions are written:

      • const square = x => x*x;

      This is a shortened way of writing a function.

      Callbacks

      • A function that is passed as an argument to another is known as a callback.
